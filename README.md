## Run Locally

Clone the project

```bash
  git clone https://github.com/HirboSH/tsx-discord-app.git
```

Go to the project directory

```bash
  cd tsx-discord-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
